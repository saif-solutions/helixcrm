# HELIXCRM PILOT USER GUIDE

## WELCOME TO HELIXCRM MVP PILOT
This is a limited pilot deployment for testing core functionality.
Please report any issues immediately.

## ACCESS
- URL: http://localhost:5173
- Test Credentials provided separately

## EXPECTED FUNCTIONALITY
1. ✅ Login/Logout
2. ✅ View Contacts
3. ✅ Add New Contacts
4. ✅ Edit Existing Contacts  
5. ✅ Delete Contacts

## KNOWN ISSUES
1. Session expires after 15 minutes (by design)
2. Mobile experience is basic
3. No advanced search/filtering
4. No email integration

## WHAT TO TEST
### Critical Tests:
1. Login with your credentials
2. Add at least 3 contacts
3. Edit a contact
4. Delete a contact
5. Logout and login again

### Security Tests (Optional):
1. Try to access after token expiry (wait 15+ minutes)
2. Verify you only see your own data

## REPORTING ISSUES
When reporting issues, please include:
1. What you were trying to do
2. What you expected to happen
3. What actually happened
4. Screenshot if possible
5. Browser console errors (F12 → Console tab)

## FEEDBACK
We welcome feedback on:
1. Usability
2. Performance
3. Missing features
4. Bug reports

## IMPORTANT NOTES
- This is TEST DATA only
- Do not enter real sensitive information
- System will be reset after pilot
- All actions are logged for security
