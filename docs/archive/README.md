# ARCHIVE DOCUMENTATION

This folder contains historical and redundant documentation that has been
superseded by the new documentation architecture.

## Structure
- `legacy-docs/` - Old scripts and utilities
- `pilot-reports/` - Pilot planning and execution documents
- `sprint-reports/` - Daily status and sprint planning documents  
- `test-reports/` - Test procedures and validation documents

## Purpose
These documents are preserved for:
- Historical reference
- Audit trail
- Learning from past decisions
- Compliance requirements

## Access Note
These documents are NOT the current source of truth. For authoritative
information, refer to `/docs/00-master/HELIX_PLATFORM_FLAGSHIP.md`

## Archive Policy
- Documents moved here are read-only
- No new documents should be added directly
- Quarterly review for permanent deletion of outdated content
